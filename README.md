# Weatherly — Smart Weather Dashboard

A responsive weather dashboard built using JavaScript and OpenWeatherMap API.

## Features
- Real-time weather search
- 5-day forecast
- Animated gradient UI
- Glassmorphism design
- Dark/Light mode toggle
- Async/Await API handling

## Tech Stack
- HTML
- CSS (Glassmorphism + Animations)
- JavaScript (ES6+)
- OpenWeatherMap API

## Setup

1. Clone the repository
2. Add your OpenWeather API key inside script.js:

   const apiKey = "YOUR_API_KEY_HERE";

3. Open index.html in browser
